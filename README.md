# A05 Care Manager Agent

AI-powered care manager agent for prioritizing and briefing on patient queues.

## Project structure

```
app/          Streamlit UI
src/          Core logic (db, ranking, context, agent)
data/         SQLite database
docs/         Architecture and design docs
tests/        Unit tests
```

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app/main.py
```
